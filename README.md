# Ecommerce Microservices Java Project

This project demonstrates multiple Java concepts using an E-commerce domain.

## Technologies Used
- Java Generics
- Lambda Expressions
- Java Streams
- JavaFX GUI
- JDBC Database
- Networking (InetAddress & URLConnection)
- Socket Programming

## Project Structure

MainApp.java – JavaFX GUI application  
Product.java – Base class  
Electronics.java / Clothing.java – Subclasses  
Repository.java – Generic collection manager  
LambdaStreamService.java – Lambda & Stream operations  
DatabaseService.java – JDBC database connection  
NetworkService.java – Networking concepts  
SocketServer.java – Server program  
SocketClient.java – Client program

## Domain
E-commerce Microservices System
